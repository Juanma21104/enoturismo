# Ontología de Enoturismo (Extensión de SEGITTUR)

Este repositorio alberga el proyecto final de la asignatura de Ingeniería Ontológica de la UPM. El proyecto consiste en el desarrollo de una ontología de enoturismo diseñada como una extensión modular de la ontología de turismo de SEGITTUR, integrando vocabularios como EBACO, FoodOn y FOAF. El desarrollo siguió la metodología NeOn, usando Chowlk para el modelado visual y Protégé para el refinamiento semántico. La ontología final (`ontology.owl`) fue evaluada con OOPS! y documentada con WIDOCO.

La documentación completa está disponible en:
**[https://juanma21104.github.io/enoturismo/docs/index-es.html](https://juanma21104.github.io/enoturismo/docs/index-es.html)**
